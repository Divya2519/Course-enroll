# Skillful Learning
Skillful Learning is a responsive online course landing page developed using HTML, CSS, and JavaScript. The project demonstrates performance optimization techniques to improve website speed and user experience.
## Features
- Responsive landing page
- Course curriculum section
- Instructor profile
- Student reviews
- Pricing plans
- Optimized images
- Lazy loading
- Deferred JavaScript
- Google Lighthouse performance analysis
## Technologies Used
- HTML5
- CSS3
- JavaScript
- Google Lighthouse
## Performance Improvements
- Optimized images for faster loading
- Implemented lazy loading for images
- Deferred JavaScript execution
- Reduced render-blocking resources
- Improved page responsiveness

- Accessibility: 94
- Best Practices: 100

- Improved understanding of website performance optimization.
